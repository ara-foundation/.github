## Ara
Ara means *a space in between* in Turkmen language. It's a medium, without concretization.
Ara aims to be between humans and computers that gives power over computer to the humans.

It's composed of following technologies and solutions:
* [CascadeFund](https://cascadefund.org/) &ndash; an open-source collaboration where codebase is maintained in a decentralized way, issues routed to a dev who wrote that part of code. All paid by users as a security. Source codes: [Web app](https://github.com/ara-foundation/cascadefund)
* Maydan &ndash; manage your computer as a knowledge base. Ontology based, automated open-source apps you found on web. Your data kept locally, OSS runs inside containers. Source code: [Maydan](https://github.com/ara-foundation/maydan)
* Hyperpayment &ndash; a protocol to describe payment flow, that can be joined by any number of parties without permission into the flow. Source code: [Website](https://github.com/ara-foundation/hyperpayment-page), [Ethereum Smartcontract Implementation](https://github.com/ara-foundation/cascadefund-smartcontracts)
